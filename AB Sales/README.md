# Анализ A/B тестирования продаж

## Описание проекта

Этот проект направлен на анализ результатов A/B тестирования для оценки эффективности изменений на сайте компании, занимающейся продажей товаров. Основная цель исследования — определить, есть ли статистически значимые различия в поведении пользователей и показателях продаж между двумя группами (A и B).

## Набор данных

Набор данных содержит информацию о заказах пользователей, таких как:
- `transactionId` — идентификатор транзакции,
- `visitorId` — идентификатор пользователя,
- `date` — дата заказа,
- `revenue` — доход от заказа,
- `group` — группа, к которой относится пользователь (A или B).

## Основные этапы и цели исследования

1. **Предобработка данных**:
    - Загрузка данных и первичный осмотр.
    - Проверка данных на наличие пропусков и дубликатов.
    - Приведение данных к нужным типам.
2. **Анализ данных**:
    - Анализ распределения пользователей по группам.
    - Проверка наличия пересекающихся пользователей между группами.
3. **Исследовательский анализ данных (EDA)**:
    - Построение кумулятивных метрик.
    - Анализ аномалий и выбросов.
4. **Статистический анализ**:
    - Проверка гипотез с использованием статистических тестов.
    - Оценка значимости различий между группами по ключевым метрикам.
5. **Выводы и рекомендации**.

## Используемые библиотеки

Для выполнения проекта использовались следующие библиотеки:
- `pandas` — для работы с табличными данными,
- `numpy` — для численных вычислений,
- `matplotlib` и `seaborn` — для визуализации данных,
- `scipy` — для проведения статистических тестов.

## Выводы

1. **Статистически значимое различие по количеству заказов на пользователя**:
    - Между группами как по сырым данным, так и после фильтрации аномалий, есть статистически значимое различие по количеству заказов на пользователя.
2. **Нет статистически значимого различия по среднему чеку**:
    - Между группами как по сырым данным, так и после фильтрации аномалий, нет статистически значимого различия по среднему чеку.
3. **График относительного изменения кумулятивного среднего количества заказов на посетителя группы B к группе A**:
    - Результаты группы B лучше, чем группы A. Они зафиксировались около среднего значения на уровне ~15%.

**Рекомендации**:
- Остановить тест, признать его успешным и перейти к проверке следующей гипотезы.
- Выявлено 58 пользователей со 183 заказами, которые попали одновременно в группы A и B. Необходимо разобраться, почему так вышло и постараться не допускать подобного в дальнейшем.

## Заключение

Результаты исследования показывают, что изменения на сайте положительно сказались на количестве заказов на пользователя в группе B по сравнению с группой A. Средний чек при этом существенно не изменился. Это свидетельствует о том, что внедрение изменений было успешным и следует продолжать работать в этом направлении.
